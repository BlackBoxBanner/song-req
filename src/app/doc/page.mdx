## What is it?

This website is builded for song request. It is a simple website that allows users to request songs to the open session. 

## Getting Started

To get started, you need to follow the steps below:

1. Enter the creator website [https://song.redbiiddsun.com/creator](https://song.redbiiddsun.com/creator)
2. Register the account
3. Sign in to the account
4. Create a session
5. Share the session link to the users
